# Confidentiality

Client briefing notes state that site data should be treated in confidence and not shared outside the project team and supervisors.

Repository policy:

- Do not commit original Leeds PDFs, DWGs, DXFs, Word documents, or slide decks.
- Commit only synthetic, anonymised, or derived non-sensitive examples.
- Keep local raw data under ignored paths such as `data/raw/`.
- Use documentation to describe data issues without redistributing the source files.
